SChartTickLabelFormatter Class

A wrapper object that contains an appropriate NSFormatter for the axes for which it is formatting tick labels.

Syntax

[Foundation.Register("SChartTickLabelFormatter", true)]
public class SChartTickLabelFormatter : Foundation.NSObject

Remarks

Instances of SChartDateTimeAxis and SChartNumberAxis (and subclasses thereof) are assigned a sensible SChartTickLabelFormatter when they are initialized. Set properties on the SChartTickLabelFormatter.Formatter property to customise your formatting. You can also subclass SChartTickLabelFormatter and set your custom object to an axis to handle formatting.
