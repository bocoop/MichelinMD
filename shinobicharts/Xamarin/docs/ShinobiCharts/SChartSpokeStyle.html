SChartSpokeStyle Class

Syntax

[Foundation.Register("SChartSpokeStyle", true)]
public class SChartSpokeStyle : Foundation.NSObject

Remarks

`SChartSpokeStyle` manages the appearance of spokes for `SChartDonutSeries` and `SChartPieSeries` objects. Spokes provide an alternative way to display labels on a pie or donut chart. Labels are positioned outside the series with a line drawn from them to the center of the slice.

For spokes to be drawn, `showSpokes` must be enabled. Spokes can be enabled on a per-slice basis. Please see our [user guide](../../user_guide.html#Spokes) for a code sample. @available Standard @available Premium @see SChartSpokesView
