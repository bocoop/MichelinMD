GetKeyValueDisplayPairs Method

Returns a dictionary of key-value pairs that will be displayed in this tooltip.

Syntax

[Foundation.Export("keyValueDisplayPairsForDataPoint:onSeries:withXAxis:withYAxis:")]
public virtual Foundation.NSMutableDictionary GetKeyValueDisplayPairs (SChartData dataPoint, SChartSeries series, SChartAxis xAxis, SChartAxis yAxis)

Parameters

dataPoint
The data point that the toolip is attempting to display values for.
series
The series that dataPoint belongs to.
xAxis
The x-axis that the series is linked to.
yAxis
The y-axis that the series is linked to.

Returns

A dictionary of NSString objects that are to be displayed in this tooltip.

Remarks

For example an OHLC series with a multi-y data point will return a dictionary with four entries. The keys will be "Open", "High", "Low", "Close" as returned by the series method SChartSeries.StringFor, with corresponding values from the given data point. The default behaviour is to use the methods SChartSeries.StringFor(string, SChartData) and SChartSeries.StringFor(string, SChartData, SChartAxis) to retrieve the desired display strings for each key and each value. This means that if you only need to change the text that is displayed for each key or value it is likely less work to subclass your series and override these methods.
