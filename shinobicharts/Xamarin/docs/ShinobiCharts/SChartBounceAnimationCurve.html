SChartBounceAnimationCurve Class

This curve evaluator describes a curve where the value starts at 0, climbs past 1 and passes it, then returns to 1, creating a 'bounce' effect.

Syntax

[Foundation.Register("SChartBounceAnimationCurve", true)]
public class SChartBounceAnimationCurve : SChartAnimationCurveProtocol

Remarks
