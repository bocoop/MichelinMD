LineRenderMode Property

Whether x-axis lines and y-axis grid lines on radial charts should be curved or straight.

Syntax

[get: Foundation.Export("lineRenderMode", ObjCRuntime.ArgumentSemantic.UnsafeUnretained)]
[set: Foundation.Export("setLineRenderMode:", ObjCRuntime.ArgumentSemantic.UnsafeUnretained)]
public virtual SChartRadialLineRenderMode LineRenderMode { get; set; }

Value

Documentation for this section has not yet been entered.

Remarks

By default, this is set to SChartRadialLineRenderMode.Curved.
  • SChartRadialLineRenderMode.Curved: Lines are drawn as an arc.
  • SChartRadialLineRenderMode.Linear: Lines are drawn as straight lines between tick frequencies, in a "spider-web" style
