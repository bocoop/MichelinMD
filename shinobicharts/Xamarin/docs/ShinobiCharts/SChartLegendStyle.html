Orientation Property

The layout of the series within the legend.

Syntax

[get: Foundation.Export("orientation")]
[set: Foundation.Export("setOrientation:")]
public virtual SChartLegendOrientation Orientation { get; set; }

Value

Documentation for this section has not yet been entered.

Remarks

The options available are:
  • SChartLegendOrientation.Vertical: Configures the layout of the series within the legend to be vertical i.e legend entries are placed from top to bottom.
  • SChartLegendOrientation.Horizontal: Configures the layout of the series within the legend to be horizontal i.e. legend entries are placed from left to right, wrapping if they reach the edge of the legend

ShowSymbols Property

Defines whether symbols will be shown in the legend.

Syntax

[get: Foundation.Export("showSymbols")]
[set: Foundation.Export("setShowSymbols:")]
public virtual bool ShowSymbols { get; set; }

Value

Documentation for this section has not yet been entered.

Remarks

By default, this property is set to true (symbols will be shown). Symbols are collected from each of the chart series using the SChartLegendItem protocol. If this is set to false, each chart series will be represented in the legend by a text label only.

SymbolAlignment Property

The alignment of the symbols in the legend.

Syntax

[get: Foundation.Export("symbolAlignment")]
[set: Foundation.Export("setSymbolAlignment:")]
public virtual SChartSeriesLegendSymbolAlignment SymbolAlignment { get; set; }

Value

Documentation for this section has not yet been entered.

Remarks

The options available are:
  • SChartSeriesLegendSymbolAlignment.Left: Configures the symbols in the legend to be placed to the left of the series name.
  • SChartSeriesLegendSymbolAlignment.Right: Configures the symbols in the legend to be placed to the right of the series name.
If ShowSymbols is set to false, this property will have no effect.
