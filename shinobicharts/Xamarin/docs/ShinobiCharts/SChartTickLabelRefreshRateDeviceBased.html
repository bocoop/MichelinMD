SChartTickLabelRefreshRateDeviceBased Class

This class provides a simple way to adjust the rate at which tick labels are refreshed.

Syntax

[Foundation.Register("SChartTickLabelRefreshRateDeviceBased", true)]
public class SChartTickLabelRefreshRateDeviceBased : Foundation.NSObject, Foundation.INSCopying, ISChartTickLabelRefreshRate, IDisposable

Remarks

By default, slower devices (see list below) will skip every second call to refresh the tick marks, while faster devices will process every call. Slow devices: - iPad Mini 1 - iPad 3rd Generation and below - iPhone 4S and below
