SChartLayer Class

A ShinobiChart plot area is made up of various layers that are each responsible for different elements of the chart.

Syntax

[Foundation.Model]
[Foundation.Protocol]
[Foundation.Register("SChartLayer", true)]
public class SChartLayer : UIKit.UIView, ISChartLayer, IDisposable

Remarks

Each layer must conform to this protocol.
