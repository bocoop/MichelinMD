SChartCrosshairTooltip Class

A text label that moves with the chart crosshair

Syntax

[Foundation.Register("SChartCrosshairTooltip", true)]
public class SChartCrosshairTooltip : UIKit.UIView

See Also

SChartCrosshair

Remarks

To create a custom Tooltip - subclass this class and override the functions below. When the standard ShinobiCharts.SChartCrosshair moves position it will call the following functions in order:

SChartCrosshairTooltip.OnSettingTooltipStyle(SChartCrosshairStyle)

SChartCrosshairTooltip.OnSettingDataPoint(SChartData, SChartSeries, ShinobiChart)

SChartCrosshairTooltip.OnSettingPosition(SChartPoint, SChartCanvas)

More information about using SChartCrosshairTooltip can be found in our tutorials 'Customizing that Crosshair' (http://www.shinobicontrols.com/blog/posts/2012/07/19/customizing-that-crosshair) and 'Data streaming, crosshairs and custom tooltips in ShinobiCharts' (http://www.shinobicontrols.com/blog/posts/2013/12/04/data-streaming,-crosshairs-and-custom-tooltips-in-shinobicharts).
