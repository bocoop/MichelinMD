SChartBarColumnSeriesStyle Class

SChartBarColumnSeriesStyle manages the look of bar and column series on a chart.

Syntax

[Foundation.Register("SChartBarColumnSeriesStyle", true)]
public class SChartBarColumnSeriesStyle : SChartSeriesStyle

Remarks

Contains any properties which are common to both bar and column series.

For properties which are specific to bar series, you should use the ShinobiCharts.SChartBarSeriesStyle. For properties which are specific to column series, you should use the ShinobiCharts.SChartColumnSeriesStyle.

CornerRatio Property

Syntax

[get: Foundation.Export("cornerRatio", ObjCRuntime.ArgumentSemantic.UnsafeUnretained)]
[set: Foundation.Export("setCornerRatio:", ObjCRuntime.ArgumentSemantic.UnsafeUnretained)]
public virtual nfloat CornerRatio { get; set; }

Value

Documentation for this section has not yet been entered.

Remarks

The ratio used to calculate a corner radius which is applied to the corners on the tip of each bar or column. The corner radius is calculated by multiplying this corner ratio value against half the bar or column width. When set to 1 the tip of each bar or column is fully rounded, when set to 0 there isn't a corner radius applied. By default, this property is set to `0`.

ShowAreaWithGradient Property

Set this property to true to add a gradient to the fill inside the bar/column.

Syntax

[get: Foundation.Export("showAreaWithGradient")]
[set: Foundation.Export("setShowAreaWithGradient:")]
public virtual bool ShowAreaWithGradient { get; set; }

Value

Documentation for this section has not yet been entered.

Remarks

If the area is not filled, this property won't have any effect. When a gradient is applied to the fill, the color starts at AreaColor, and finishes at AreaColorGradient. The gradient isn't configurable. It runs from left to right on column series, and from top to bottom on bar series.

By default, this property is set to true.
