SChartBasePointStyle Class

SChartBasePointStyle is the base class for all point styles in ShinobiCharts.

Syntax

[Foundation.Register("SChartBasePointStyle", true)]
public class SChartBasePointStyle : Foundation.NSObject, Foundation.INSCopying, IDisposable

Remarks

This point style object manages the appearance of datapoint on a chart. This base class is used for the point style in the ShinobiCharts.SChartRadialLineSeriesStyle class. It is also subclassed by ShinobiCharts.SChartBaselinePointStyle which every other point style class corresponds to. For more information on these other point styles see ShinobiCharts.SChartBaselinePointStyle.
