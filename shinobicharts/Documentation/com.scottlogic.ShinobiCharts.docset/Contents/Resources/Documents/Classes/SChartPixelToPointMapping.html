pixelPoint

The pixel point, in chart coordinates.

@property (nonatomic, assign) CGPoint pixelPoint

Discussion

This is the “suggested point”.

For example, on line series this point will be the interpolated position where a trigger point intersects the line.

On a multiseries column or bar this will be offset so the point is positioned above the correct column / to the side of the correct bar.
